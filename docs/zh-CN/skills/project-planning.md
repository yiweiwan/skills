# project-planning：项目规划

## 文件位置

`.claude/skills/project-planning/SKILL.md`

## 适用场景

用于项目计划、工作拆解、路线图、里程碑、实施计划、任务清单、风险登记、估算、Sprint 计划、交付计划和跨技术业务协同。

## 能力重点

- 明确目标、范围、干系人、约束、截止日期、依赖和完成标准。
- 把工作拆成可验证的交付物。
- 按依赖和风险排序，而不只是按模块排序。
- 暴露假设、待决策事项、阻塞和未知风险。


```text

```

