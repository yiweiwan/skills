# document-summarization：文档总结

## 文件位置

`.claude/skills/document-summarization/SKILL.md`

## 适用场景

只用于总结、提取、重组或综合用户提供的文档、笔记、访谈、报告、PDF、会议记录、政策或长文本。不用于开放式网页调研。

## 能力重点

- 保留来源含义、名称、日期、数字、义务和决策。
- 按摘要、重点、行动项、风险、开放问题等结构输出。
- 有页码、章节或时间戳时保留引用。
- 标记不清楚、缺失或矛盾的来源内容。

## 示例提问

```text
请把这份长文档总结成执行摘要、风险和行动项。
```
