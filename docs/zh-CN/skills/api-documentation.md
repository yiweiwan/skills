# api-documentation：API 文档

## 文件位置

`.claude/skills/api-documentation/SKILL.md`

## 适用场景

只用于 API 文档、端点参考、REST 文档、GraphQL 文档、SDK 文档、认证说明、请求响应示例、错误码、限流、分页和集成指南。

## 能力重点

- 先查看真实路由、schema、handler、OpenAPI、GraphQL schema、SDK 或测试。
- 写清认证、端点、参数、请求体、响应、错误、限制和示例。
- 示例要贴近真实使用。
- 保持字段名、状态码和错误结构与实现一致。

## 示例提问

```text
请为这个订单 API 写一份外部开发者文档。
```
