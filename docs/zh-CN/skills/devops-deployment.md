# devops-deployment：DevOps 与部署

## 文件位置

`.claude/skills/devops-deployment/SKILL.md`

## 适用场景

用于部署、CI/CD、Docker、容器、云配置、环境变量、构建流水线、发布流程、基础设施即代码、监控、日志、回滚计划和生产就绪检查。

## 能力重点

- 先检查脚本、Dockerfile、compose、CI workflow 和部署配置。
- 明确构建、测试、打包、迁移、发布、健康检查和回滚步骤。
- 不把密钥写进源码或日志。
- 保持部署改动小而可验证。

```text
请为这个项目补一个 Dockerfile 和 docker-compose。
```

```text
帮我修复 GitHub Actions 构建失败的问题。
```

```text
请给这次数据库迁移设计发布和回滚步骤。
```
