# competitor-analysis：竞品分析

## 文件位置

`.claude/skills/competitor-analysis/SKILL.md`

## 适用场景

只用于竞品、产品、供应商或市场选项对比，包括定位、功能、价格、目标用户、差异点、GTM、优劣势、风险和推荐矩阵。

## 能力重点

- 先明确比较目标、用户、市场和决策标准。
- 只比较会影响决策的维度。
- 区分可观察事实和分析判断。
- 给出适用于当前场景的推荐，而不是泛泛排名。

## 示例提问

```text
请比较 Notion、Confluence 和 Linear 的团队知识管理能力。
```
