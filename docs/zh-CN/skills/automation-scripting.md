# automation-scripting：自动化脚本

## 文件位置

`.claude/skills/automation-scripting/SKILL.md`

## 适用场景

用于创建脚本、命令行工具、批处理任务、文件处理工具、数据转换脚本、本地开发自动化、PowerShell、Bash、Python、Node 脚本、定时任务和可重复的运维流程。

## 能力重点

- 明确输入、输出、运行环境和失败影响。
- 优先沿用仓库已有脚本语言和命令风格。
- 编写可重复执行、输入校验清晰、错误信息明确的脚本。
- 对危险操作使用 dry-run、显式参数或确认机制。

## 示例提问

```text
请写一个 PowerShell 脚本，批量整理这些日志文件并输出汇总 CSV。
```

```text
帮我写一个 Node 脚本，把 JSON 数据转换成前端需要的格式。
```

```text
请把这个重复的手工构建流程自动化。
```
