# frontend-design：前端设计

## 文件位置

`.claude/skills/frontend-design/SKILL.md`

## 适用场景

用于设计或优化 UI、布局、视觉层级、响应式表现、设计系统、仪表盘、工具类产品、网站、表单和交互式前端体验，也适用于 UI/UX 评审、视觉打磨、可访问性、组件设计和产品前端体验优化。

## 能力重点

- 先观察现有应用的组件、样式和设计约定。
- 根据用户、任务、信息密度和设备限制设计界面。
- 保持现有产品语言，除非明确要求重新设计。
- 检查移动端和桌面端的文本溢出、重叠和交互状态。

## 示例提问

```text
请优化这个后台页面的布局，让信息更容易扫描。
```

```text
帮我设计一个订单管理仪表盘，保持专业简洁。
```

```text
请检查这个页面在移动端的可用性问题。
```
