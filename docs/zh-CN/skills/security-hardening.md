# security-hardening：安全加固

## 文件位置

`.claude/skills/security-hardening/SKILL.md`

## 适用场景

用于安全评估和加固，包括认证、授权、密钥管理、输入校验、依赖漏洞、Web 安全头、注入风险、访问控制、隐私和安全编码。

## 能力重点

- 明确资产、角色、信任边界、敏感数据和入口点。
- 追踪输入如何进入存储、渲染、命令、外部服务和日志。
- 在服务端靠近数据访问处做授权。
- 优先使用框架自带安全能力和成熟库。

