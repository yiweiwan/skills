# fullstack-integration：全栈联调

## 文件位置

`.claude/skills/fullstack-integration/SKILL.md`

## 适用场景

用于打通前后端端到端行为，包括 API 契约、生成客户端、登录态、cookie、CORS、文件上传下载、实时更新、表单提交、乐观 UI 和跨层功能实现。

## 能力重点

- 从 UI 事件一路追踪到请求、接口、服务、数据库、响应和 UI 更新。
- 保持字段名、错误结构和契约在前后端一致。
- 服务端校验权威，前端校验用于提升体验。
- 检查认证、CSRF、CORS、cookie 和 token 刷新。

## 示例提问

```text
请把这个前端表单接到后端创建接口，并处理错误状态。
```

```text
登录后 cookie 没带上，请帮我排查前后端配置。
```

```text
请设计文件上传的前后端完整流程。
```
