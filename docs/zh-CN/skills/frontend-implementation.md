# frontend-implementation：前端实现

## 文件位置

`.claude/skills/frontend-implementation/SKILL.md`

## 适用场景

仅作为通用前端实现的总控/路由 skill 使用。React 使用 `react-frontend`，Vue 使用 `vue-frontend`，Next.js 使用 `nextjs-frontend`，前端性能使用 `frontend-performance`。

## 能力重点

- 先识别框架、包管理器、构建脚本、路由、样式系统和测试方式。
- 能使用更窄框架 skill 时优先使用更窄 skill。
- 遵循现有组件、hook、store 和数据请求模式。
- 明确 loading、empty、error、disabled、success 等状态。
- 优先使用语义化 HTML 和已有组件库。
