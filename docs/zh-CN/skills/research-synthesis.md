# research-synthesis：研究汇总

## 文件位置

`.claude/skills/research-synthesis/SKILL.md`

## 适用场景

仅作为研究类任务的总控/路由 skill 使用。网页调研使用 `web-research`，竞品或产品对比使用 `competitor-analysis`，已有文档总结使用 `document-summarization`，论文和学术资料综述使用 `literature-review`。

## 能力重点

- 先明确研究问题、决策背景、受众和置信度要求。
- 判断任务属于网页调研、竞品分析、文档总结、论文综述还是混合研究。
- 能使用更窄 skill 时优先使用更窄 skill。
- 区分事实、来源观点、分析、假设和建议。

## 示例提问

```text
请比较这三款产品，输出选型建议和风险。
```

```text
把这些调研资料整理成一页管理层简报。
```

```text
请总结这几篇文章的共同结论和分歧。
```
