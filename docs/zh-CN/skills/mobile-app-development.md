# mobile-app-development：移动端开发

## 文件位置

`.claude/skills/mobile-app-development/SKILL.md`

## 适用场景

用于 React Native、Expo、Flutter、Swift、Kotlin、Android、iOS、移动端 UI、导航、原生模块、设备 API、权限、离线状态、推送通知和应用商店准备。

## 能力重点

- 明确平台、框架、导航、样式、原生模块和构建工具。
- 关注小屏、触摸目标、安全区、键盘、离线和弱网。
- 处理权限拒绝、App 生命周期、深链和推送通知。
- 保持 Android 与 iOS 的差异清晰可见。


```text

```
