# Document Summarization

## Workflow

1. Identify document type, audience, desired length, and output format.
2. Preserve source meaning, names, dates, numbers, obligations, and decisions.
3. Separate summary, key points, action items, risks, open questions, and quotes when relevant.
4. Keep page, section, or timestamp references when the source provides them.
5. Mark unclear, missing, or contradictory source content.

## Output Patterns

- Executive summary.
- Decision log.
- Action item list.
- Risk and issue list.
- Thematic synthesis.

## Verification

- Do not invent facts not present in the source.
- Check names, dates, totals, owners, and quoted wording.
- Note source limitations.
