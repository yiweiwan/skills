# Email Communication

## Workflow

1. Identify the communication type and audience.
2. Switch to a narrower skill whenever possible.
3. Use this skill only for mixed communication tasks or tone calibration.
4. Make ownership, deadlines, decisions, and next steps explicit.
5. Adjust tone for power dynamics, sensitivity, culture, and relationship history.

## Prefer Narrow Skills

- Email or reply: `email-drafting`.
- Status or executive update: `stakeholder-update`.
- Meeting recap: `meeting-notes`.
- Release announcement: `release-notes`.

## Verification

- Check names, dates, attachments, links, deadlines, and promised actions.
- Make sure the message matches the requested tone and channel.
