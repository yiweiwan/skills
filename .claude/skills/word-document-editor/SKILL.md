# Word Document Editor

## Workflow

1. Identify document type, audience, tone, required format, and preservation constraints.
2. Preserve headings, numbering, defined terms, names, dates, cross-references, and obligations.
3. Improve clarity, consistency, concision, and flow without inventing facts.
4. Keep edits compatible with Word-style formatting.
5. Summarize changes when editing an existing document.

## Editing Rules

- Do not rewrite legal, HR, financial, or compliance language beyond the requested scope.
- Mark assumptions, missing information, and placeholders.
- Use tables only when they improve comparison, ownership, or tracking.

## Verification

- Check names, dates, numbering, references, tables, and unresolved placeholders.
- Confirm the final text matches the requested tone and audience.
