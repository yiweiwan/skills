# Spreadsheet Dashboard

## Workflow

1. Identify audience, KPI definitions, filters, refresh cadence, and source data.
2. Separate raw data, calculations, and dashboard presentation.
3. Choose charts and tables that match the decision being made.
4. Keep metric definitions and date ranges visible.
5. Design for refresh without breaking formulas or charts.

## Rules

- Avoid overcrowded dashboards.
- Use consistent number formats, date periods, and labels.
- Flag stale data and manual refresh requirements.

## Verification

- Cross-check dashboard totals against source data.
- Test filters, pivots, and chart ranges.
